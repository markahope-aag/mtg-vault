# MTG Vault

Personal Magic: The Gathering inventory and Commander deckbuilding tool.

Desktop-first web app. Single user. Built on Next.js 15 + Supabase + Drizzle, deployed to Vercel.

## What it does

- **Inventory** — track what cards you own, where they are, what you paid, what they're worth now
- **Deckbuilder** — build Commander decks against your inventory with own/need diffs
- **Bracket engine** — compute the official Commander Bracket (1–5) for any deck, with a "remove these to drop a bracket" diff
- **Value tracking** — total collection value, per-deck value, value-over-time charts
